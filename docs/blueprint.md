# **App Name**: Vosiqs

## Core Features:

- Vibe-to-Playlist Conversion: Accept a natural language prompt and convert it into a curated playlist of YouTube videos.
- AI-Powered Curation: Use the Gemini API as a tool to interpret prompts, identify moods, and generate queries for the YouTube Data API. Personalize results based on user history and trending videos.
- Smart Playlist Metadata: Automatically generate a unique name and descriptive summary for each playlist.
- Video Playback and Saving: Allow users to watch videos within the app and save playlists to their YouTube accounts.
- Playlist Re-roll: Implement functionality to re-generate new videos based on the original prompt.
- Free Tier with Ads: Offer a free tier with limited features (max 30 videos per playlist, 3 re-rolls, limit 10 prompts in a 5-hour window, max 20 playlists saved) and ad support.
- Pro Subscription: Offer a Pro subscription ($7.99) with unlimited prompts, videos, advanced mood filters, and playlist downloads.

## Style Guidelines:

- Primary color: Vibrant red (#F44336) to align with the original image's logo color.
- Background color: Very light gray (#F5F5F5), to enhance the bright text and video previews in the interface. Desaturated near 20% saturation and bright near 95% lightness.
- Accent color: A reddish-orange (#FF5722), used for interactive elements to highlight the ability to try the tool.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral user interface, with flexibility between sizes and weights.
- Centered text input area with ambient background, vibe card layout for video grid, sidebar for logged-in users, and a banner for the premium upsell.
- Clean and simple icons for user interface elements like 'save,' 're-roll,' and 'open in YouTube.'
- Subtle hover effects on video thumbnails and smooth transitions for playlist updates.
